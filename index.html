Jaxbot's Vim Tutorial

By Jonathan Warner

1. Download Vim

Windows: gVim builds from tuxproject.de
Mac OSX: MacVim or brew
Ubuntu/Debian: sudo apt-get install vim-gtk

2. Grab a minimal vimrc

vimrc is your set of configuration commands.
Here's a minimal but easily expandable vimrc and vim dotfiles.
Download this and put the .vim folder at ~/.vim on OSX/Linux, or C:\Users\You\vimfiles on Windows.

3. Time to launch!

Open up Vim. It should look something like this:
If it has a white background or toolbars, your vimrc is not being loaded correctly.
Make sure ~/.vim/vimrc is the file you downloaded.

4. Learn Vim

Now that we have Vim ready to use, it's time to learn Vim.
Vim is a modal editor with different commands from what you may be used to.
These resources will be a great help:
Open Vim Tutorial - Interactive tutorial to learn movement/editing
Vim Adventures - Interactive tutorial game
Vim Commands Cheat Sheet - List form
Graphical Vim Keyboard Chart

You should be able to get the basic insert, delete, undo, write motions down relatively soon.
Check back here once you're ready for more.
