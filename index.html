Jaxbot's Vim Tutorial

By Jonathan Warner

1. Download Vim

Windows: gVim builds from tuxproject.de
Mac OSX: MacVim or brew
Ubuntu/Debian: sudo apt-get install vim-gtk

2. Grab a minimal vimrc

vimrc is your set of configuration commands.
It is read from ~/.vimrc or C:\Users\You\vimrc
Nerds argue about how much of a boilerplate you should have.
Here's a medium one; not too fat, not too slim.
Add fat or leanness as you see fit.
